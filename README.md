# Information-Gathering Tool

InfoGa - This tool is an open source software Information Gathering Tool supported with api Hackertarget & threatcrowd using this we can easily get information to their websites.

# Features :

Features :

- Traceroute
- Test Ping
- DNS lookup
- Reverse DNS
- Host Record
- Subdomain Finder
- Find DNS server
- Zone transfer
- Whois Lookup
- Geo IP Lookup
- Reverse IP Lookup
- TCP Port Scan
- Subnet Lookup
- ASN Lookup
- HTTP Header
- Extract Page Link
- Reverse Analytics Search
- IP Scanner
- Web IP Finder
